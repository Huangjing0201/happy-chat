* 运行方式：
```bash
# clone项目到本地
git clone https://github.com/Huangjing0201/happy-chat.git

# 安装项目依赖
npm install

# 运行以下命令打开浏览器的localhost:8888查看
cd happy-chat
node server.js 或 npm start
```
